# 👨🏻‍💻 Teste a aplicação

  ### Clone o repositório
```
  git clone https://github.com/vleonecezar/take-control.git
```
  ### Instale todos os pacotes
```
  npm install
```
  ### Abra a aplicação localmente
```
  npm start
```
  ### Deixe a aplicação pronta para produção
```
  npm run build
```
